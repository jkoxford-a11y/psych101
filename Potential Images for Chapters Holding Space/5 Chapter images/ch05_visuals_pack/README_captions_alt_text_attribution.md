# Chapter 5 visual files

## `fig_scn_circadian_pathway`

**Caption:** Light resets the circadian clock through a pathway from the retina to the suprachiasmatic nucleus (SCN), which helps regulate pineal melatonin signaling and sleep timing. The key point is that light and melatonin shift circadian timing rather than simply turning sleepiness on or off.

**Alt text:** Flow diagram showing bright light leading to the retina, then to the SCN clock in the hypothalamus, then to pineal signaling, then to the melatonin rhythm. A teaching note explains that light, darkness, and melatonin mainly shift circadian timing, while caffeine works more on sleep pressure.

**Attribution:** Original educational diagram created for Jon Oxford’s Psych101 textbook project. No external image source used. No empirical data plotted.

## `fig_tolerance_withdrawal_cycle`

**Caption:** Repeated drug use can produce neuroadaptation. Over time, the same dose may have a smaller effect, and stopping or reducing use may produce withdrawal symptoms. Using again may then be reinforced because it relieves withdrawal, even when the original desired effect is reduced.

**Alt text:** Circular feedback-loop diagram showing repeated use leading to brain adaptation, tolerance, withdrawal after reducing use, and use that relieves withdrawal. A highlighted arrow labels this as negative reinforcement because relief can make reuse more likely. A note distinguishes tolerance from withdrawal.

**Attribution:** Original educational diagram created for Jon Oxford’s Psych101 textbook project. No external image source used. No empirical data plotted.
