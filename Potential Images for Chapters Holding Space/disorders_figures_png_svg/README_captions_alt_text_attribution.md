## Design notes

- 16:9 slide-friendly layout.
- Figure numbers are intentionally not embedded in the images, so the assets can be reordered or reused.
- PNG files are ready for slides. SVG files preserve editable vector elements and text where supported.
- The diathesis-stress diagram uses a shared threshold and vulnerability as baseline load. This resolves the common ambiguity in the prompt: the same stressor has different consequences depending on vulnerability.
- The AI comparison uses slightly tightened wording: “No therapist-client privilege,” “limited clinical continuity,” and “not a crisis-response system” avoid overclaiming across products and jurisdictions.
