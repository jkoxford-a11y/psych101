# Prologue diagram assets

Original educational diagrams generated for the General Psychology textbook prologue. No external images or datasets used.

## learning_exposure_retrieval_decision_tree

**Suggested use:** Prologue, after the Opening Case or near the retrieval-practice section.

**Caption:** Exposure means information was present. Learning means it changed what you can retrieve, explain, apply, or judge later.

**Alt text:** Decision tree showing that rereading and recognition create familiarity, while recall and application provide stronger evidence of learning.

**Attribution:** Original diagram by Jon Oxford / ChatGPT, 2026. CC BY 4.0 recommended if publicly shared.

## ai_replacement_vs_scaffolding

**Suggested use:** Prologue AI section.

**Caption:** AI supports learning when it follows effort and feedback. It weakens learning when it replaces the first retrieval attempt.

**Alt text:** Two-column diagram comparing AI use that replaces student retrieval with AI use that critiques and strengthens a student’s own explanation.

**Attribution:** Original diagram by Jon Oxford / ChatGPT, 2026. CC BY 4.0 recommended if publicly shared.
