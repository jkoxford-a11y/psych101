# Chapter 5 candidate external visuals

Staging area for free-to-use visuals found online. These are **not yet promoted** into `docs/images/ch05/`; promotion should happen only after Jon decides the asset is instructional enough to use in the textbook or slide library.

Selection rule for this folder: include source URL, license, attribution text, candidate use, and a recommendation. Favor public domain or simple CC BY assets. Treat CC BY-SA assets as usable but slightly more encumbered because adaptations may need to be shared under a compatible license.

## Candidate 1 — Circadian rhythm / SCN light pathway

![Circadian rhythm labeled — NIH/NIGMS public domain](https://upload.wikimedia.org/wikipedia/commons/5/55/Circadian_rhythm_labeled.jpg)

- **Suggested local filename if promoted:** `ch05-circadian-rhythm-scn-light-pathway-nih.jpg`
- **Source page:** <https://commons.wikimedia.org/wiki/File:Circadian_rhythm_labeled.jpg>
- **Direct file:** <https://upload.wikimedia.org/wikipedia/commons/5/55/Circadian_rhythm_labeled.jpg>
- **Source/creator:** National Institute of General Medical Sciences, NIH.
- **License/status:** Public domain; U.S. federal government/NIH work, according to the Wikimedia Commons file page.
- **Attribution text:** Circadian rhythm diagram. National Institute of General Medical Sciences, NIH. Public domain.
- **Candidate use:** Chapter 5, Section 1, circadian rhythms and SCN entrainment by light.
- **Instructional value:** Moderate. Useful as a simple visual anchor for light → SCN → physiological/behavioral rhythms. It is small and not as clean as an original diagram, but public-domain status makes it safe.
- **Recommendation:** Keep as a candidate. Best use is slides or as a temporary placeholder; an original 16:9 two-process/circadian diagram would be better for final textbook use.

## Candidate 3 — Standard hypnogram of normal sleep

![Hypnogram — Wikimedia Commons CC BY-SA 3.0](https://upload.wikimedia.org/wikipedia/commons/b/b7/HYPNOGRAM_created_by_Natasha_k.jpg)

- **Suggested local filename if promoted:** `ch05-hypnogram-normal-sleep-cc-by-sa.jpg`
- **Source page:** <https://commons.wikimedia.org/wiki/File:HYPNOGRAM_created_by_Natasha_k.jpg>
- **Direct file:** <https://upload.wikimedia.org/wikipedia/commons/b/b7/HYPNOGRAM_created_by_Natasha_k.jpg>
- **Source/creator:** Own work by Wikimedia Commons user Tash510.
- **License/status:** Creative Commons Attribution-ShareAlike 3.0 Unported (CC BY-SA 3.0), according to the Wikimedia Commons file page.
- **Attribution text:** Hypnogram by Tash510, Wikimedia Commons, CC BY-SA 3.0.
- **Candidate use:** Chapter 5, Section 2, sleep architecture across the night.
- **Instructional value:** Low-to-moderate as-is. It is a recognizable hypnogram, but it shows NREM stage 4 separately; our chapter uses the current NREM 1/2/3 framing. This mismatch would confuse students.
- **Recommendation:** Do **not** use directly in the chapter. Use as a reference for designing an original corrected hypnogram with Wake/REM/N1/N2/N3 only.
