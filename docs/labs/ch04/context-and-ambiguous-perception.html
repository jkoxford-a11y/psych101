What the comparison means

Bottom-up input constrains perception, but prior knowledge and context can bias which interpretation wins when the input is ambiguous. This demonstrates a top-down influence. It does not prove that expectation determines all perception; an unambiguous stimulus would leave much less room for context to change the result.
