      4. What the Pattern Means

      Concept Reveal

      Input below threshold produces no action potential. Once threshold is crossed, the neuron produces a stereotyped spike. Stimulus strength is usually represented by how frequently neurons fire and how many neurons participate—not by making each spike proportionally larger.

      This is a foundational teaching model. Real spikes vary somewhat across cells and biological conditions; “all-or-nothing” does not mean every spike in nature is mathematically identical.
