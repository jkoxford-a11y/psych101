# Chapter 4 Good Continuation Figures

## ch04_good_continuation_smooth_paths.png / .svg

**Caption:** Good continuation: when strokes overlap, the visual system tends to organize them into the smoothest continuous paths rather than abrupt fragments. The same input can be parsed in multiple ways, but smooth continuation is usually favored.

**Alt text:** Three-panel diagram illustrating good continuation. The first panel shows ambiguous overlapping curved strokes. The second panel highlights two smooth continuous paths as the favored perceptual grouping. The third panel shows an abrupt grouping that is physically possible but perceptually less likely.

**Attribution:** Original diagram created for Oxford Psychology 101, 2026. No external copyrighted image assets used.

**Public-sharing status:** Safe for public sharing.

**Accuracy note:** Schematic demonstration of the Gestalt principle of good continuation; not an empirical measurement.

## ch04_good_continuation_wm_redrawing.png / .svg

**Caption:** Good continuation can override familiar letter identity. When W- and M-like strokes overlap, the visual system tends to group the smoothest continuous paths into a new pattern rather than preserving the original letters.

**Alt text:** Two-panel diagram showing overlapping W- and M-like strokes. The first panel emphasizes the familiar letter forms. The second panel fades the letters and highlights two smooth crossing continuations, showing how perception can reorganize the same strokes into a new pattern.

**Attribution:** Original explanatory redrawing created for Oxford Psychology 101, 2026. Concept inspired by Wertheimer’s Gestalt demonstrations of good continuation; no modern source image copied or traced.

**Public-sharing status:** Safe for public sharing.

**Accuracy note:** Historically inspired demonstration, not a facsimile of Wertheimer’s original 1912 figure.
