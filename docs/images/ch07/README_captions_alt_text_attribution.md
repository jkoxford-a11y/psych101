# Chapter 7 Figure Metadata

## Figure 7.4 — Reinforcement schedules

- **Files:** `fig_reinforcement_schedules.svg` (editable source); `fig_reinforcement_schedules.png` (2400 × 1680 rendered asset).
- **Caption:** Reinforcement schedules shape established response patterns, while reinforcement history can influence persistence after reinforcement stops. Variable-ratio and variable-interval schedules commonly produce relatively steady responding. Fixed-ratio schedules produce post-reinforcement pauses followed by rapid runs, whereas fixed-interval schedules can produce pauses followed by accelerating responding. These cumulative records illustrate characteristic shapes rather than universal rate rankings. The extinction comparison illustrates a common partial-reinforcement extinction effect: responding can decline more slowly after intermittent than continuous reinforcement when response rate is evaluated relative to each condition’s own baseline. Neither panel shows speed of initial acquisition.
- **Alt text:** Four color-coded columns pair reinforcement-rule diagrams with cumulative-response records. Fixed ratio shows reinforcement after a set response count and a post-reinforcement pause followed by a rapid run. Variable ratio shows reinforcement after unpredictable response counts and high, relatively steady responding. Fixed interval shows reinforcement for the first response after each set interval and a cumulative scalloped record with pauses followed by acceleration. Variable interval shows reinforcement for the first response after changing intervals and steady, moderate responding. A smaller extinction graph normalizes each condition to 100 percent of its own pre-extinction response-rate baseline; illustrative points show responding declining more gradually after an intermittent than a continuous reinforcement history. The figure distinguishes established response pattern from extinction persistence and explicitly states that acquisition speed is not depicted.

- **Interpretation caveats:** Panel A depicts established responding while reinforcement is available, not speed of initial acquisition and not a universal quantitative ranking of response rates. Exact rates depend on schedule values, reinforcement density, organism, task, and history. Panel B illustrates a common partial-reinforcement extinction effect after normalizing each history to its own baseline; it does not support a universal ordering among fixed-ratio, variable-ratio, fixed-interval, and variable-interval schedules.
