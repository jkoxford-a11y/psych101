## `ch05_consciousness_state_content_access_framework.png`

**Instructional purpose:** Establish the chapter's organizing framework: conscious state, conscious content, and conscious access are three separate questions rather than one overall level.

**Required embedded text:**

- CONSCIOUS STATE
- Awake and capable of experience?
- CONSCIOUS CONTENT
- What is being experienced?
- CONSCIOUS ACCESS
- What can be reported or used flexibly?
- These can change separately.

**Caption:** Consciousness is not one overall level. This chapter separates three questions: the organism's conscious state, the contents of experience, and the information available for report or flexible use. These dimensions interact but can change in different combinations.

**Alt text:** Three-column diagram labeled Conscious State, Conscious Content, and Conscious Access. The columns ask whether the organism is awake and capable of experience, what is currently experienced, and what information can be reported or used flexibly. A statement beneath the columns says that these dimensions can change separately.

## `ch05_experience_report_researcher_inference.png`

**Instructional purpose:** Show that producing a report adds cognitive and response operations, while both report and no-report methods require researchers to infer experience from indirect evidence.

**Required embedded text:**

- POSSIBLE EXPERIENCE
- REPORT PATH
- ATTEND
- DECIDE
- RESPOND
- REPORT
- NO-REPORT PATH
- EYE / PUPIL / PHYSIOLOGY
- INDICATOR
- RESEARCHER INFERENCE
- Neither route directly reads experience.

**Caption:** Experience is not the same as reporting experience. Producing a report adds attention, decision, and response requirements. No-report methods reduce some of those demands by using physiological or behavioral indicators, but both approaches require researchers to infer experience from indirect evidence.

**Alt text:** Two pathways begin with a possible experience and converge on researcher inference. The upper report path passes through attention, decision, response, and report. The lower no-report path passes through an eye, pupil, or physiological measure and an indicator. A statement beneath the diagram says that neither route directly reads experience.

**Source/creation method:** Original figure generated for Oxford Psychology 101 with ChatGPT image generation, 2026. No external copyrighted image assets intentionally used.

**License:** Project-owned generated original.

**Attribution:** Original figure generated for Oxford Psychology 101 with ChatGPT image generation, 2026. No external copyrighted image assets intentionally used.

**Public-sharing status:** Safe for public sharing pending final visual, conceptual, spelling, and accessibility review.

**Use status:** Pending image arrival and approval; intended for the no-report methods box.

**Accuracy caveats:**

- A report is evidence about experience, not the experience itself.
- No-report methods do not directly measure consciousness.
- Eye movements, pupil changes, and physiological signals do not always indicate conscious experience.
- Do not depict one path as scientifically superior or more certain.
- Both paths must converge on researcher inference; neither should bypass it.
- Do not depict a researcher directly observing or looking inside subjective experience.

**Design constraints:** Standalone 16:9 landscape figure; clean white or very light background; one left-to-right diagram with two labeled horizontal paths; large readable typography; minimal text; clean arrows; no title, figure number, caption, alt text, source line, attribution, status label, production note, medical examples, detailed brain anatomy, logo, or watermark inside the artwork.
